# Badge

**Storybook:** `Components/Badge`

## Overview

Small label or count. Use for status, categories, counts, or tags. Supports variants: default, secondary, destructive, outline, ghost, link.

## When to use

- Status pills (New, Beta, Deprecated), category tags, or numeric counts (e.g. notifications).
- Next to titles or in table cells.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `span hlmBadge` | Badge; set `variant` and optional `size`. |

## Variants & options

- **variant:** `default` | `secondary` | `destructive` | `outline` | `ghost` | `link`.
- **size:** typically default or dot-only for status.
