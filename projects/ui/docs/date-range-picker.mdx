# Date Range Picker

**Storybook:** `Components/Date Range Picker`

## Overview

Range date picker: select start and end date. Supports captionLayout, min/max, disabled, formatDates, transformDates, and autoCloseOnEndSelection. Use with forms via ngModel or formControlName.

## When to use

- Date ranges (report period, booking check-in/out, filter from–to).
- When you need both start and end date.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-date-range-picker` | Root; bind range (start/end), min, max, etc. |

## Variants & options

- **formatDates** / **transformDates**; **autoCloseOnEndSelection**.
- Use with `ngModel` or `formControlName` for range value.
