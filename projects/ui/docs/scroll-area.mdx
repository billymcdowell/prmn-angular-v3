# Scroll Area

**Storybook:** `Components/Scroll Area`

## Overview

Styled wrapper around `ng-scrollbar` for consistently themed scroll areas. Use for long lists, code blocks, or scrollable panels inside cards and dialogs.

## When to use

- Any overflow area where you want consistent scrollbar styling (e.g. sidebar nav, modal body, code block).
- When default browser scrollbars don’t match the theme.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-scroll-area` | Wrapper; content inside scrolls with themed scrollbar. |

## Variants & options

- Horizontal/vertical; optional visibility (always, hover, etc.) depending on ng-scrollbar.
