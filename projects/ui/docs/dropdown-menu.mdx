# Dropdown Menu

**Storybook:** `Components/Dropdown Menu`

## Overview

Dropdown menu with items, labels, shortcuts, separators, checkboxes, radio groups, and submenus. Trigger opens a positioned menu.

## When to use

- Action menus (e.g. row actions, "More" menu), profile menu, or any list of actions opened by click.
- When you need shortcuts, icons, submenus, or checkable/radio items.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-dropdown-menu` | Root (or use as content for context menu). |
| `button hlmDropdownMenuTrigger` | Trigger. |
| `*hlmDropdownMenuPortal` | Portal. |
| `hlm-dropdown-menu-content` | Panel. |
| `hlmDropdownMenuItem` | Item. |
| Labels, shortcuts, separators, checkboxes, radio groups, submenus | As needed. |

## Variants & options

- Supports submenus, disabled items, and custom content (e.g. avatar + name in trigger).
