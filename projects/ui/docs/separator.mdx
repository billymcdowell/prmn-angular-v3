# Separator

**Storybook:** `Components/Separator`

## Overview

Visual divider between content. Use orientation="horizontal" (default) for a horizontal line, orientation="vertical" for a vertical line. Set decorative="true" when the separator is purely visual and not meaningful to screen readers.

## When to use

- Between sections (e.g. in a dropdown, card, or sidebar).
- Vertical divider between icon and text, or between buttons.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-separator` | Divider; set `orientation`, `decorative`. |

## Variants & options

- **orientation:** `horizontal` | `vertical`.
- **decorative:** `true` when divider is visual only (reduces verbosity for screen readers).
