# Showcase

**Storybook:** `Examples/Showcase`

## Overview

Visual showcase of the most commonly used UI components in the library, grouped by category. Use as a reference for composition and patterns.

## When to use

- Quick reference when building a new page: see which components exist and how they look together.
- Copy-paste starting points for forms, cards, tables, and navigation.

## Contents

- **Typography** — headings, body, muted, code, lists.
- **Buttons & badges** — variants, sizes, icons.
- **Cards** — header, content, footer, actions.
- **Form controls** — input, textarea, select, checkbox, switch, radio.
- **Layout** — tabs, alerts, tooltips, avatar, separator.
- **Data** — table, pagination, breadcrumb, scroll area.
- **Overlays** — dropdown, popover, dialog, sheet, command.
- **Feedback** — progress, spinner, skeleton, empty, sonner.
- **Pickers** — date picker, calendar, combobox, autocomplete.
- **Navigation** — sidebar, navigation menu.
