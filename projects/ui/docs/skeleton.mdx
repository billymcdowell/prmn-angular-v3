# Skeleton

**Storybook:** `Components/Skeleton`

## Overview

Placeholder that shows a pulsing animation while content is loading. Use for cards, avatars, text lines, or custom shapes. Respects prefers-reduced-motion (motion-safe:animate-pulse).

## When to use

- Loading states for cards, lists, or profile (avatar + lines).
- When you want to reserve space and show structure before content loads.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-skeleton` | Placeholder block; use as div or override element. |

## Variants & options

- Use multiple skeletons to mimic layout (e.g. circle for avatar, rectangles for text).
- Respects prefers-reduced-motion.
