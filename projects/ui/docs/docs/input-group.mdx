# Input Group

**Storybook:** `Components/Input Group`

## Overview

Input with addons: text, icons, or buttons before/after. Use hlm-input-group; add hlm-input-group-addon (with align), hlm-input-group-input, hlm-input-group-text, or hlm-input-group-button.

## When to use

- Search with magnifier icon; input with unit ($, kg); input with clear or submit button.
- When you need a single visual block (input + addons).

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-input-group` | Container. |
| `hlm-input-group-addon` | Addon slot (align start/end). |
| `hlm-input-group-input` | The input. |
| `hlm-input-group-text` | Text addon. |
| `hlm-input-group-button` | Button addon. |

## Variants & options

- **align** on addon for positioning; combine text, icon, or button.
