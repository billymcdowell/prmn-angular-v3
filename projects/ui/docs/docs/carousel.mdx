# Carousel

**Storybook:** `Components/Carousel`

## Overview

Carousel/slider built on Embla. Use `hlm-carousel-content` and `hlm-carousel-item` for slides. Optional: `hlm-carousel-previous`, `hlm-carousel-next`, `hlm-carousel-slide-display`. orientation: `horizontal` or `vertical`.

## When to use

- Image/content sliders, testimonials, or featured content that cycles.
- When you need prev/next controls and optional indicators.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-carousel` | Root; set `orientation="horizontal" \| "vertical"`. |
| `hlm-carousel-content` | Slide container. |
| `hlm-carousel-item` | Each slide. |
| `hlm-carousel-previous` | Previous button. |
| `hlm-carousel-next` | Next button. |
| `hlm-carousel-slide-display` | Optional slide counter/display. |

## Variants & options

- **orientation:** `horizontal` | `vertical`.
