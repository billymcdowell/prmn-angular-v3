# Data Table Layout

**Storybook:** `Examples/Layouts/DataTable`

## Overview

Data table layout with toolbar, filters, bulk actions, and pagination that combines table, form controls, and badges.

## When to use

- List/table views with search, filters, column selection, bulk actions (delete, export), and pagination.
- When you need a full data-grid pattern.

## Key components used

- [Table](table.mdx) — data grid.
- [Input](input.mdx), [Select](select.mdx), [Button](button.mdx) — toolbar and filters.
- [Badge](badge.mdx) — counts, status.
- [Pagination](pagination.mdx) — paging.
- [Dropdown Menu](dropdown-menu.mdx) — row actions, bulk actions.
- [Checkbox](checkbox.mdx) — row selection.
