# Spinner

**Storybook:** `Components/Spinner`

## Overview

Loading indicator. Use icon (default: lucideLoader), aria-label for accessibility, and size via CSS (e.g. class). Respects prefers-reduced-motion.

## When to use

- Button or inline loading state (e.g. "Saving..."), or full-page loading.
- When you need a simple circular spinner.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-spinner` | Spinner; set aria-label, size via class. |

## Variants & options

- **size** via CSS/class; **aria-label** for screen readers; respects prefers-reduced-motion.
