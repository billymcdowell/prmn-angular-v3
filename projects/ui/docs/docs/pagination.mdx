# Pagination

**Storybook:** `Components/Pagination`

## Overview

Pagination nav with previous/next, page links, and ellipsis. Use hlmPagination, hlmPaginationContent, hlmPaginationItem, hlmPaginationLink, hlm-pagination-previous, hlm-pagination-next, hlm-pagination-ellipsis. For full numbered pagination with page size, use hlm-numbered-pagination.

## When to use

- Tables, lists, or search results that span multiple pages.
- When you need prev/next and optional page numbers.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `nav hlmPagination` | Root. |
| `hlmPaginationContent` | Container for items. |
| `hlmPaginationItem` | Wrapper for link or ellipsis. |
| `hlmPaginationLink` | Page link. |
| `hlm-pagination-previous` | Previous button. |
| `hlm-pagination-next` | Next button. |
| `hlm-pagination-ellipsis` | Ellipsis when pages omitted. |
| `hlm-numbered-pagination` | Full numbered + page size. |

## Variants & options

- Build from links and prev/next; or use numbered pagination for page size selector.
