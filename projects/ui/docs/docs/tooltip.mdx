# Tooltip

**Storybook:** `Components/Tooltip`

## Overview

Hover (or focus) trigger shows a small popover with text. Use hlmTooltip="Your text" on the trigger. Supports position (top, bottom, left, right), showDelay, hideDelay, and tooltipDisabled.

## When to use

- Short labels for icons, truncated text, or extra context on hover/focus.
- When content is one short line; use [Hover Card](hover-card.mdx) for richer content.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlmTooltip="text"` on trigger | Tooltip content. |
| Trigger element | Element that receives hover/focus. |

## Variants & options

- **position:** top | bottom | left | right. **showDelay**, **hideDelay**, **tooltipDisabled**.
