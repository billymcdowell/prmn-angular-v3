# Slider

**Storybook:** `Components/Slider`

## Overview

Range input as a track with one or more thumbs. Supports min, max, step, single value or range, vertical orientation, ticks, and disabled state. Use with forms or reactive value binding.

## When to use

- Numeric range selection (volume, price range, opacity) or single value (e.g. 0–100).
- When you need precise control and optional tick marks.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-slider` (or Brn slider + Hlm styling) | Root; bind value(s), min, max, step. |

## Variants & options

- **min**, **max**, **step**; single value or range (two thumbs); **orientation:** horizontal | vertical; **disabled**; optional ticks.
