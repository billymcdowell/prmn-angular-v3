# Dashboard Layout

**Storybook:** `Examples/Layouts/Dashboard`

## Overview

Dashboard layout example showing a header, filters, stats cards, table, and pagination composed from multiple UI components.

## When to use

- App dashboard: header, sidebar or top nav, filter bar, KPI cards, data table, and pagination.
- When you need a template that combines layout + data + actions.

## Key components used

- [Sidebar](sidebar.mdx) or header — navigation.
- [Card](card.mdx) — stats/KPI cards.
- [Input](input.mdx), [Select](select.mdx), [Button](button.mdx) — filters.
- [Table](table.mdx) — data grid.
- [Pagination](pagination.mdx) — paging.
- [Breadcrumb](breadcrumb.mdx) — optional.
- [Dropdown Menu](dropdown-menu.mdx) — row or header actions.
