# Typography

**Storybook:** `Components/Typography`

## Overview

Text styles for headings, body, and inline elements. Use hlmH1–hlmH4, hlmP, hlmLead, hlmLarge, hlmSmall, hlmMuted, hlmBlockquote, hlmCode, hlmUl.

## When to use

- Consistent heading and body styles across pages and cards.
- Lead paragraph, small/muted text, blockquotes, code, and lists.

## Key elements / API

| Directive | Purpose |
|-----------|---------|
| `hlmH1` … `hlmH4` | Heading levels. |
| `hlmP` | Body paragraph. |
| `hlmLead` | Lead/large intro text. |
| `hlmLarge`, `hlmSmall` | Size variants. |
| `hlmMuted` | Muted color text. |
| `hlmBlockquote` | Quote. |
| `hlmCode` | Inline/code. |
| `hlmUl` | Unordered list. |

## Variants & options

- Apply to appropriate elements (h1–h4, p, blockquote, code, ul) for semantics.
