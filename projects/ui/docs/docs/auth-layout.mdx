# Auth Layout

**Storybook:** `Examples/Layouts/Auth`

## Overview

Authentication layout example with a centered card, form fields, alerts, and secondary actions demonstrating composition of multiple primitives.

## When to use

- Sign in, sign up, forgot password, or reset password pages.
- When you need a centered, single-column form with title, fields, primary button, and links.

## Key components used

- [Card](card.mdx) — centered container.
- [Typography](typography.mdx) — title and description.
- [Form Field](form-field.mdx), [Input](input.mdx), [Button](button.mdx) — form and actions.
- [Alert](alert.mdx) — optional error/info message.
- [Label](label.mdx), [Separator](separator.mdx) — structure.
