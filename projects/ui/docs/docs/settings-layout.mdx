# Settings Layout

**Storybook:** `Examples/Layouts/Settings`

## Overview

Account settings layout combining forms, switches, checkboxes, cards, alerts, and tabs to showcase a realistic preferences page.

## When to use

- User/account settings: profile, notifications, security, billing.
- When you need forms, toggles, and sectioned layout (tabs or cards).

## Key components used

- [Tabs](tabs.mdx) — Account / Password / Billing (or similar).
- [Card](card.mdx) — section containers.
- [Form Field](form-field.mdx), [Input](input.mdx), [Textarea](textarea.mdx) — form rows.
- [Switch](switch.mdx), [Checkbox](checkbox.mdx) — toggles and options.
- [Alert](alert.mdx) — success/error message.
- [Button](button.mdx) — Save, Cancel.
