# Switch

**Storybook:** `Components/Switch`

## Overview

Toggle switch for boolean state. Supports checked (model), disabled, and accessibility attributes (id, aria-label, aria-labelledby, aria-describedby). Works with reactive forms.

## When to use

- On/off settings (notifications, dark mode, feature flags).
- When you want a switch instead of a checkbox for a single boolean.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-switch` | Switch; use with ngModel or formControlName. |

## Variants & options

- **disabled**; **aria-label** or **aria-labelledby** for accessibility.
