# Item

**Storybook:** `Components/Item`

## Overview

List item / card row with optional media, title, description, footer, and actions. Variants: default, outline, muted. Sizes: default, sm.

## When to use

- List rows (search results, notifications, file list) or card-like rows with thumbnail, title, description, and actions.
- When you need consistent layout for repeated items.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-item` | Root; set `variant`, `size`. |
| Media, title, description, footer, actions | Slots as needed. |

## Variants & options

- **variant:** `default` | `outline` | `muted`.
- **size:** `default` | `sm`.
