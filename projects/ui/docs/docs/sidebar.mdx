# Sidebar

**Storybook:** `Components/Sidebar`

## Overview

App sidebar with optional collapse, groups, and theming. Uses sidebar tokens (--sidebar, --sidebar-foreground, etc.). Compose with sidebar trigger, content, header, footer, and nav items.

## When to use

- Main app navigation (nav links, sections, user block at bottom).
- When you need collapsible sidebar and consistent sidebar tokens.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-sidebar` | Root. |
| Trigger, content, header, footer | Structure. |
| Nav items, groups | Content. |

## Variants & options

- Collapsible state; use sidebar theme tokens for colors (see [THEMING-TOKENS](THEMING-TOKENS.mdx)).
