# Button Group

**Storybook:** `Components/ButtonGroup`

## Overview

Groups buttons (or other controls) with shared borders. Use orientation `horizontal` or `vertical`. Optional: `hlm-button-group-separator`, `hlm-button-group-text`.

## When to use

- Toolbars (e.g. bold, italic, underline), segmented controls, or multiple related actions that should look like one control.
- When you want first/middle/last styling and no double borders between items.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `div hlm-button-group` | Container; set `orientation="horizontal"` or `"vertical"`. |
| `hlm-button-group-separator` | Visual divider between items. |
| `hlm-button-group-text` | Non-button label inside group. |

## Variants & options

- **orientation:** `horizontal` | `vertical`.
