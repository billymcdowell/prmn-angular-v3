# Icon

**Storybook:** `Components/Icon`

## Overview

Icon directive for ng-icon. Use `hlm` or `hlmIcon` on ng-icon and set size: xs | sm | base | lg | xl | none, or a custom CSS length.

## When to use

- Consistent icon sizing next to labels, in buttons, or in lists.
- When using @ng-icons (e.g. Lucide) and want theme-aligned sizes.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `ng-icon hlm` / `ng-icon hlmIcon` | Themed icon; set `size`. |

## Variants & options

- **size:** `xs` | `sm` | `base` | `lg` | `xl` | `none`, or custom CSS length.
- Use with `provideIcons({ ... })` for icon set.
