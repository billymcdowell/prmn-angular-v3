# Date Picker Multi

**Storybook:** `Components/Date Picker Multi`

## Overview

Multi date picker: select multiple dates. Supports captionLayout, min/max, minSelection, maxSelection, disabled, formatDates, transformDates, and autoCloseOnMaxSelection. Use with forms via ngModel or formControlName.

## When to use

- Selecting multiple dates (e.g. available days, recurring dates).
- When user must pick more than one date.

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-date-picker-multi` | Root; bind dates array, min/max selection, etc. |

## Variants & options

- **minSelection** / **maxSelection**; **formatDates** / **transformDates**; **autoCloseOnMaxSelection**.
- Integrate with forms.
