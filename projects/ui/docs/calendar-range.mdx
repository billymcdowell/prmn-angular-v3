# Calendar Range

**Storybook:** `Components/CalendarRange`

## Overview

Date range calendar. Select start and end date. Same options as Calendar: captionLayout, min/max, disabled, dateDisabled, weekStartsOn, defaultFocusedDate.

## When to use

- Range selection (e.g. "from – to"). Typically used inside [Date Range Picker](date-range-picker.mdx).

## Key elements / API

| Element / directive | Purpose |
|--------------------|---------|
| `hlm-calendar-range` | Range calendar; bind range, min, max, etc. |

## Variants & options

- Same as [Calendar](calendar.mdx); selection is a range (start/end).

- [Calendar](calendar.mdx) — single date.
